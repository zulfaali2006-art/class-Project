Overview

TransiTera is a simple Java-based system that simulates a mediator between passengers and travel companies.
The project focuses on applying Data Structures concepts using a clear object-oriented design.


---

Purpose

Demonstrate understanding of data structures.

Implement data structures manually without using Java Collections.

Apply OOP principles in a realistic scenario.



---

Data Structures Used

All data structures are implemented from scratch:

Dynamic Array

Linked List

Stack

Queue

Simple Key–Value Map


Each structure is used where it logically fits within the system.


---

Main Components

TransiTera: Core system mediator.

TravelCompany: Manages trips.

Trip: Represents travel routes.



---

Technologies

Java

Object-Oriented Programming

Custom Data Structures

Notes

This project is for educational purposes and is not connected to a database or external libraries.
